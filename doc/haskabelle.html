Haskabelle

What is Haskabelle?

Haskabelle is a converter from Haskell source files to Isabelle/HOL theories implemented in Haskell itself.

Obtaining Haskabelle.

The current Haskabelle release is available here for download:

haskabelle.tar.gz

This also includes documentation to be found in the [FIXME shell?] doc/ direcorty.

Attention! Until the release of Isabelle 2009, Haskabelle is nothing else than a preview tool – it won't work with Isabelle 2008. To test it anyway, consider using an appropropriate Isabelle developement snapshot, e.g. ...
